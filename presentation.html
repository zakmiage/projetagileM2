Solution Soutenance M2 MIAGE

Une vision claire.
Zéro double saisie.

Génération FSDIE automatisée
Fini la mise en forme manuelle. Les dossiers FSDIE sont générés en PDF avec toutes les pièces jointes, prêts à être déposés.
Maîtrise budgétaire en temps réel
Le budget se met à jour automatiquement. Comparez le prévisionnel au réel instantanément.
Coordination fluide des équipes
Un planning interactif pour répartir les bénévoles. Visualisez les créneaux et évitez les conflits d'horaires.
Stack Technique : Angular 17, Express.js, MySQL.
Méthodologie Agile, 42 tests d'intégration, Git Flow.
